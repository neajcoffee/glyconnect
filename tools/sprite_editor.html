    ℹ️ Comment ça marche
    

      Chaque ligne est encodée comme un octet où les N bits de poids fort
      (N = largeur du sprite) représentent les pixels. Le reste est zéro.
      


      Exemple ligne ░░█░░ (largeur 5) :
      

      → 00100 en binaire MSB → décalé à gauche dans un octet : 0010 0000 = 0x20.
      


      C'est le format Adafruit GFX standard, identique pour les glyphes de police (3 bits utilisés)
      et pour les bitmaps custom (jusqu'à 8 bits).
