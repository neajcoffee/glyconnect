Glyconnect — Outils

Petits outils web pour préparer des designs de matrice LED 32×8 et lire les frames du firmware.

🎨

Decor Share

Dessine une image fixe 32×8 et exporte le code RGB565 prêt à coller dans le firmware.

 🚀

Decor Boot

Bannière défilante (largeur jusqu'à 256 px) pour l'écran de démarrage. Aperçu animé inclus.

 ✏️

Sprite Editor

Éditeur de sprites avancé avec gestion multi-frames.

 📖

Reader

Visualise et décode les frames RGB565 envoyées par le firmware.
